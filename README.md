# MOZI

## How To Start

### Node version

`v16.15.1`

### MySQL version

`v8.0.29`

How to install

- [MySQL downloads](https://dev.mysql.com/downloads/mysql/)

#### Develop Mode

```
yarn install

yarn dev
```

### Production Mode

```
yarn build

yarn start
```

### Test Mode

```
yarn test
```
